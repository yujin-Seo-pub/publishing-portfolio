안녕하세요 책임감 있는 웹퍼블리셔 서유진입니다.
웹퍼블리싱이라는 새로운 분야에 도전하게 되어 꾸준히 성장하고 있습니다.
저에게 코딩은 부지런한 삶을 살 수 있게 해주고, 좋아해서 더 잘하기 위해 노력하게 되는 일입니다.
꼼꼼하고 세심한 작업으로 사용자의 불편함을 최소화할 수 있는 결과물을 지향하며,
그 과정에서 다양한 위치의 팀원들을 존중하고 소통하며 원활한 협업이 될 수 있게 제 역할을 다하겠습니다.
한발 더 나아가기 위해 계속해서 노력하며 배우고 탐구하는 자세를 갖춘 웹퍼블리셔가 되겠습니다.

Skill

HTML 웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 해오고 있습니다. 특히 웹 표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃 설계합니다.
CSS CSS 코딩은 크로스 브라우징(CrossBrowsing)을 위해 안정적인 CSS 속성을 사용해서 HTML 요소를 배치하고 디자인합니다. 특별한 경우 CSS3에서 새롭게 지원하는 속성을 활용해서 좀 더 다양한 CSS 배치와 디자인을 합니다.
JS 네비게이션, 탭 메뉴, 어코디언 등 웹 퍼블리싱에 반드시 필요한 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수한 제이쿼리 코딩으로 제작합니다. 동적인 모션을 위한 플러그인이 필요한 경우에는 플러그인도 활용이 가능합니다.
ResponsiveWeb 반응형 웹 제작에 있어서 크로스 브라우징(Cross Browsing)을 위해 안정적인 기존 방식인 float, position, Media Query를 활용한 반응형 웹 제작 경험을 충분히 가지고 있습니다. 필요에 따라 플렉스(Flex)방식으로 반응형 웹사이트 제작도 충분히 가능합니다.
PhotoShop 웹사이트 UI 디자인 작업은 포토샵을 활용하며 웹 UI 디자인 작업을 위한 간단한 포토샵 편집이 가능합니다.
TextEditor 웹퍼블리셔에게 가장 중요한 텍스트 에디터는 메인으로 Visual Studio Code를 사용합니다.
